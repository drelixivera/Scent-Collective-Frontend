🛍️ Scent-Collective Frontend
A premium, high-performance e-commerce frontend designed for a curated collection of scents. This project is a demonstration of modern frontend engineering, focusing on UI/UX, responsive design, and efficient data handling without heavy backend dependencies.
🚀 Live Demo
https://scent-collective-frontend.vercel.app
🛠️ Tech Stack
1. Frontend Framework: React.js
2. Styling: Bootstrap 5 & Custom CSS3
3. Animations: Custom Fade-In Hooks
4. Deployment: Vercel
5. Icons/UI: React-Bootstrap & Lucide-style Icons

✨ Key Features
1. Performance Optimized: Transitioned from a MERN stack to a standalone frontend for instant load times and 100% uptime.
2. Advanced Filtering: Real-time search and filtering by Category (Fresh, Woody, Floral, Luxury) and Price Range.
3. Responsive Design: Fully optimized for mobile, tablet, and desktop views—perfect for the "Ogbomoso lifestyle" on the go.
4. Smooth UX: Integrated smooth-scroll navigation and custom fade-in animations for a premium feel.

📸 The Collection
The product catalog is powered by a localized data architecture, allowing for:
1. Zero Latency: No waiting for API calls.
2. Offline Support: Basic browsing functionality remains intact even with spotty connectivity.
3. Scalability: Easily extendable product-data structure.

📂 Project Structure
├── public/          # Static assets (Images, Icons)
├── src/
│   ├── data/        # Localized Product Data (JSON/JS)
│   ├── components/  # Reusable UI Components
│   ├── Styles/      # Custom CSS
│   └── useFadeIn.js # Custom Animation Hook
└── package.json     # Project Dependencies

👷 Project Setup
To run this project locally:
1.  Clone the repo: git clone [https://github.com/drelixivera/Scent-Collective-Frontend.git](https://github.com/drelixivera/Scent-Collective-Frontend.git)
2.  Install dependencies: npm install
3.  Start the development server: npm start
